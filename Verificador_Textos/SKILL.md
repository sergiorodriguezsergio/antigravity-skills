# Skill: Verificador de Textos

Esta Skill se encarga de contrastar textos con información real en la web, prestando especial atención a nombres propios y datos factuales.

## Instrucciones de Uso

1. **Búsqueda y Corrección**:
   - Busca en la web información para corregir el texto proporcionado.
   - Presta atención máxima a los nombres propios.
   - Si no hay fuentes fiables, comunícalo. **JAMÁS inventes nada (alucinaciones)**.

2. **Flujo de Trabajo**:
   - **Paso 1**: Enumera las correcciones propuestas.
   - **Paso 2**: Proporciona enlaces a las fuentes consultadas.
   - **Paso 3**: Una vez que el usuario dé el "OK", entrega el texto corregido tal cual, con las negritas oportunas y sin las fuentes.

3. **Reglas de Formato**:
   - Si solo se tiene nombre y un apellido, **no propongas el segundo apellido**.
   - No incluyas coletillas tipo "por confirmar" si se pide omitir un punto. Deja la información original sin contrastar en ese caso.
   - Respeta el estilo original del texto.
